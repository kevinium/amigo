## About The Project

<img width="1673" alt="Home Page" src="https://user-images.githubusercontent.com/56315179/125462340-984e9369-a526-42c8-af98-e7bb0c3cdeb4.png">

Amigo is a new way to connect to Friends, Family and even attend to meetings at work, all in one place

Why Amigo:
* Minimal design app, to not confuse you about complicated usage.
* Straight forward features, to get your work done efficiently.
* Reduced Clutter : So that everyone from a 5 year old kid to his grandpa find it easy to use :smile:


### Built With
* [JavaScript](https://www.javascript.com)
* [NodeJS](https://nodejs.org/en/)
* [webRTC](https://webrtc.org)
* [socket.io](https://socket.io)
* [EJS](https://ejs.co)
* [JQuery](https://jquery.com)
* [Figma](https://www.figma.com) _for UI prototyping_
